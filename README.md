# IOC-M3-EAC6_OlavMartos
Exercici d'avaluació continua nº6, per el Modul nº3 del curs de IA y ML d'IOC.

## Introducció
**En qué consisteix aquest projecte?**
Tracta de fer un projecte Python de software complet, no sols utilitzant el model sino també utilitzant
llibreries com *pylant*, *pydocs*, *MLFlow*, etc.

El projecte tracta de trobar quatre clusters per a un dataset de ciclistes. El dataset es genera 
automáticament mitjançant el script ./generardataset.py
